# 🚀 AI Resume Analyzer & Builder

An AI-powered Resume Analyzer & Builder built using the MERN stack. This application helps users analyze resumes, get intelligent feedback, and generate professional resumes using AI.

---

## 🌟 Features

✨ AI Resume Analysis  
🧠 Smart Suggestions & Improvements  
📝 Resume Builder with Live Preview  
🔐 User Authentication (Login/Register)  
📂 Resume Upload & Processing  
📈 Interview Preparation Module  
⚡ Fast & Responsive UI  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Vite
- SCSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**AI Integration:**
- Google Generative AI (Gemini API)
